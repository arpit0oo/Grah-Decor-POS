# Grah Decor POS

## Project Description
Grah Decor POS is a Point of Sale (POS) system designed for small to medium-sized businesses in the home decor industry. It streamlines sales operations, inventory management, and customer relations to enhance business efficiency and customer satisfaction.

## Features
- **Sales Management**: Easily process transactions and manage sales.
- **Inventory Tracking**: Keep track of stock levels and receive notifications for low inventory.
- **Customer Management**: Maintain customer records to improve relationships and tailor marketing efforts.
- **Reporting & Analytics**: Generate detailed sales reports and analytics for better decision-making.
- **User-Friendly Interface**: Intuitive and easy-to-use interface for quick onboarding of staff.

## Tech Stack
- **Frontend**: React.js
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Deployment**: Render

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/arpit0oo/Grah-Decor-POS.git
   cd Grah-Decor-POS
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables:
   Create a `.env` file and include the necessary configurations.
4. Run the application:
   ```bash
   npm start
   ```
